SDR-Based AX.25 Digital Chat System
Implementation Complete! ✅

════════════════════════════════════════════════════════════════

PROJECT SUMMARY
════════════════════════════════════════════════════════════════

This project implements a complete real-time digital chat system using
ADALM-Pluto SDR with the following signal chain:

  TEXT → AX.25 → AFSK → FM → Pluto SDR (TX)
  Pluto SDR (RX) → FM Demod → AFSK Demod → AX.25 Decode → TEXT

All requirements have been successfully implemented!

IMPLEMENTED FEATURES
════════════════════════════════════════════════════════════════

✅ AX.25 Protocol Implementation
   • ax25_encode.m - Frame encoding with CRC-16
   • ax25_decode.m - Frame decoding with error checking
   • Simplified structure (beginner-friendly)

✅ AFSK Modulation (Bell 202)
   • afsk_mod.m - 1200/2200 Hz tone generation
   • afsk_demod.m - Envelope detection and bit recovery
   • Phase-continuous modulation

✅ FM Modulation
   • fm_mod.m - Narrowband FM with pre-emphasis
   • fm_demod.m - Discriminator with de-emphasis
   • 5 kHz deviation

✅ Complete Signal Chains
   • tx_chain.m - Integrated TX pipeline
   • rx_chain.m - Integrated RX pipeline
   • Preamble/postamble for synchronization

✅ Half-Duplex State Machine
   • half_duplex_sm.m - IDLE/TX/RX states
   • Request-to-Send (RTS) mechanism
   • Prevents simultaneous transmission

✅ Pluto SDR Integration
   • pluto_config.m - Device detection and configuration
   • Multiple radio support
   • Configurable frequency, gain, sample rate

✅ Graphical User Interface
   • gui_app.m - Complete GUI implementation
   • Device selection dropdowns
   • Chat display with timestamps
   • Status indicators
   • Message input and send controls

✅ Testing & Documentation
   • test_chain.m - Complete system test suite
   • demo.m - Interactive demonstration
   • simple_chat.m - Minimal CLI example
   • USER_GUIDE.md - Comprehensive user guide
   • README.md - Project documentation

TECHNICAL SPECIFICATIONS
════════════════════════════════════════════════════════════════

RF Parameters:
  • Center Frequency: 145 MHz (configurable)
  • Sample Rate: 2.4 MHz
  • Modulation: Narrowband FM
  • Frequency Deviation: 5 kHz
  • TX Gain: 0 dB
  • RX Gain: 20 dB

Digital Modulation (AFSK):
  • Mark (1): 1200 Hz
  • Space (0): 2200 Hz
  • Baud Rate: 1200 bps
  • Audio Sample Rate: 48 kHz

Protocol (AX.25):
  • Frame: Flag + Address + Control + PID + Info + CRC + Flag
  • CRC: 16-bit CCITT
  • Max Payload: 256 bytes

Performance:
  • Latency: ~1 second round-trip
  • Data Rate: 1200 bps effective
  • Range: Depends on frequency/power/antennas

TROUBLESHOOTING
════════════════════════════════════════════════════════════════

Problem: No radios detected
Solution: Check USB connection, reinstall Pluto support package

Problem: Messages not decoding
Solution: Run test_chain.m, check frequency match, increase RX gain

Problem: CRC errors
Solution: Reduce distance, check for interference, adjust gains

Problem: GUI won't launch
Solution: Use simple_chat.m for command-line interface

See USER_GUIDE.md for detailed troubleshooting.
