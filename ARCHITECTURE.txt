DATA FLOW EXAMPLES
═══════════════════════════════════════════════════════════════════

TRANSMIT:
  "Hello" → [01001000 01100101 01101100...] → AX.25 Frame
  → [1 0 1 1 0 0...] bits → [1200Hz 2200Hz 1200Hz...] tones
  → AFSK Audio → FM Modulated I/Q → RF @ 145 MHz

RECEIVE:
  RF @ 145 MHz → Complex I/Q → FM Demod → Audio
  → [1200Hz 2200Hz...] tones → [1 0 1...] bits
  → AX.25 Frame → [01001000...] bytes → "Hello"


KEY PARAMETERS
═══════════════════════════════════════════════════════════════════

RF:     145 MHz center, 2.4 MHz sample rate, 5 kHz deviation
AFSK:   1200/2200 Hz, 1200 baud, 48 kHz audio rate
AX.25:  Frame with CRC-16, max 256 byte payload
Timing: ~1 sec latency, half-duplex operation
