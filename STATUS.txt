TROUBLESHOOTING:
════════════════════════════════════════════════════════════════════

Problem: GUI still won't start
  → Try gui_simple instead of gui_app
  → Use simple_chat for CLI interface
  → Check MATLAB version (R2019b+ for gui_app)

Problem: Buttons don't respond
  → Close and reopen GUI
  → Try gui_simple (more reliable)
  → Check MATLAB console for errors

Problem: No radios detected
  → Check USB connections
  → Verify PlutoSDR Support Package installed
  → Try different USB port
  → Restart MATLAB

Problem: Can't send messages
  → Verify radios are connected
  → Check Request TX button is clicked
  → Ensure status shows TX mode
  → Verify send button is enabled
